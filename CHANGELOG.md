## [0.1.0] - 2026-03-14

### Added
- Initial npm package release
- Claude Code plugin with `/markviewz` skill
- GitHub-flavored markdown rendering (tables, task lists, strikethrough, autolinks)
- YAML frontmatter support (collapsed by default)
- Dark mode (auto-follows system)
- Print support (Cmd+P)
- CLI wrapper (`markviewz file.md`)
- Drag-and-drop file opening
- Local image support (relative paths)
- Single-instance reuse via `open -a`
